World Models Without High-Definition Maps.

research note / 2026.06

← Back to research blog

High-definition maps are useful when the world behaves like yesterday. They become less useful when road work appears overnight, lane markings fade, temporary signs move, or a human driver creates a situation the map never described.

A world model for driving should treat the map as one possible memory source, not as the center of intelligence. The live scene must remain primary: geometry, motion, intent, visibility, surface condition, and the causal effect of the vehicle's own actions.

What the model needs to know

The model needs to maintain a compact state that predicts what matters next. It should know which objects can move, which free space is likely to remain free, which signals are reliable, and how uncertainty should change the planned maneuver.

This is different from memorizing routes. The system must learn local physical regularities and use them under pressure, especially when the map is incomplete or contradicted by the sensor stream.

Why this matters

Map-free intelligence is not only a deployment convenience. It is a useful test for whether a model understands the world as an active system. If the model can adapt in a new district, a temporary construction zone, or an unmapped road, it has learned more than a static prior.
